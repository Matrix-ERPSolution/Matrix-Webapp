아이디 : 6~16자 영문, 숫자
	만족x -> 6~16자의 영소문자, 숫자만 사용 가능합니다.
	만족0 -> 사용 가능한 아이디입니다.
	중복 -> 사용 중인 아이디입니다. 다른 아이디를 입력해주세요.


비밀번호 : 6~16자 영문 대 소문자, 숫자, 특수문자
	만족x -> 	6~16자의 영문 대 소문자, 숫자, 특수문자만 사용 가능합니다.
	만족o ->
	안전성 -> 숫자 ->  6~16자의 영문 대 소문자, 숫자, 특수문자만 사용 가능합니다.
		영문 -> 6~16자의 영문 대 소문자, 숫자, 특수문자만 사용 가능합니다.
		영문+숫자 -> 사용하실 수 있는 비밀번호입니다. (안전도 등급 적정)
		 영문+숫자+특수문자 -> 사용하실 수 있는 비밀번호입니다. (안전도 등급 높음)
		공백들어갔을때 -> 비밀번호 내 공백은 사용할수 없습니다.

비밀번호 확인 : 
	일치x : 비밀번호와 비밀번호 확인이 일치하지 않습니다.
	일치o : 정상 입력 되었습니다.

이름 :
	만족x (길이,숫자,특수문자) : 2~12자의 한글, 영문만 사용 가능합니다.
